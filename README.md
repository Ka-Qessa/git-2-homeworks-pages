***Немного фактов о Масяне***

Масяня это самый обычный кот без породы. Его шерсть пушистая, остается везде, где он полежит. Он - отличный охотник на мышей, кротов и птиц. 

Что он любит? 
* есть;
* спать; 
* гулять. 

Больше никаких забот в его жизни нет.
